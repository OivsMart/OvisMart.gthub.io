


OvisMart is an online marketplace connecting local & international artisans, sellers, and buyers for unique products (fashion, electronics, accessories, interior decor, cars, jewelry, shoes, beauty, health, food, etc.)

    Features
    - Browse products by category
    - Search for items
    - Sellers profile

  Live Demo https://ovismart.github.io/

Tech Stack: HTML, CSS, JavaScript, GitHub Pages

  Getting Started
    1. Clone repo: `git clone https://github.com/OvisMart/OvisMart.github.io.git`
    2. Open `index.html 
`
